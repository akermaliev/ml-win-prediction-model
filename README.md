# ML Win Prediction Model

This project builds a machine learning model to predict outcomes and identify key performance drivers using competitive match data.

## Overview
- Dataset: Competitive match data
- Models: Logistic Regression, KNN
- Accuracy: ~97–98%

## Key Insights
- Gold economy is the strongest predictor of winning
- Early-game advantages heavily influence outcomes
- Objective control (dragons, towers) is critical

## Tech Stack
- Python (pandas, scikit-learn)
- Jupyter Notebook
